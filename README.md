# Countdown Timer

## Description
A responsive Countdown Timer web application built using HTML, CSS, Bootstrap, JavaScript, and jQuery.

## Features
- Live countdown timer
- Responsive design
- Simple and clean user interface
- Easy to use

## Technologies Used
- HTML5
- CSS3
- Bootstrap
- JavaScript
- jQuery

- index.html
- style.css
- script.js
